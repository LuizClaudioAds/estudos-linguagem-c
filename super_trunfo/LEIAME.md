# Super Trunfo de Países – Cadastro de Cartas (Nível Novato)

Este projeto foi desenvolvido como parte do desafio da disciplina Introdução à Programação de Computadores, do curso de Análise e Desenvolvimento de Sistemas.

O objetivo deste nível é criar a base de um jogo estilo Super Trunfo, permitindo o cadastro e a exibição de duas cartas contendo informações sobre cidades.

---

## Objetivo do Programa

O programa permite que o usuário cadastre duas cartas do Super Trunfo, informando dados como:

- Estado
- Código da carta
- Nome da cidade
- População
- Área
- PIB
- Número de pontos turísticos

Após o cadastro, o sistema exibe na tela todas as informações de forma organizada e legível.

---

## Tecnologias Utilizadas

- Linguagem C
- GitHub Codespaces
- Compilador GCC

---

## Funcionalidades Implementadas

- Leitura de dados via teclado
- Armazenamento das informações em variáveis apropriadas
- Exibição formatada dos dados das duas cartas cadastradas

---

## Restrições do Desafio (Nível Novato)

Conforme especificado no desafio:

- O programa cadastra apenas duas cartas
- Não são utilizadas estruturas de repetição (`for`, `while`)
- Não são utilizadas estruturas de decisão (`if`, `else`)
- O foco é exclusivamente em entrada, processamento e saída de dados
